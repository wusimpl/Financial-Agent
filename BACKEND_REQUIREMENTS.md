# 后端需求清单

说明：本清单根据当前前端页面整理，只覆盖界面需要后端提供的数据和能力，不包含深浅色切换、面板展开、侧栏收起等纯界面交互。

## 股票检索与自选股

| 需求名称 | 需求描述 |
| --- | --- |
| 股票搜索 | 支持按股票代码或公司名称搜索股票。 |
| 股票基础信息 | 返回股票代码、公司名称、公司标识等展示信息。 |
| 自选股列表 | 返回用户关注的股票列表及每只股票的基础信息。 |
| 当前股票切换 | 根据选中的股票代码返回对应页面数据。 |

## 个股概览

| 需求名称 | 需求描述 |
| --- | --- |
| 最新报价 | 返回当前价格、涨跌额和涨跌幅。 |
| 交易状态 | 返回盘前、开盘、盘后、隔夜、休市等市场状态。 |
| 日内行情 | 返回开盘价、最高价、最低价和昨收价。 |
| 成交数据 | 返回当前成交量和平均成交量。 |
| 估值指标 | 返回市值、市盈率、Beta、每股收益和目标价。 |
| 财务日历 | 返回下一次财报日期。 |
| 股息信息 | 返回下一次派息日、除息日和股息率。 |
| 价格区间 | 返回 52 周价格区间。 |

## 行情图表

| 需求名称 | 需求描述 |
| --- | --- |
| 时间范围行情 | 支持 1D、5D、1M、3M、6M、YTD、1Y、5Y、All 等时间范围。 |
| K 线数据 | 返回每个交易周期的开盘价、最高价、最低价、收盘价。 |
| 成交量序列 | 返回每个交易周期的成交量。 |
| 均线指标 | 返回 MA20、MA50、MA200 指标。 |
| RSI 指标 | 返回 RSI(14) 指标。 |
| MACD 指标 | 返回 MACD、信号线和柱状值。 |

## SEC 文件与财务数据

| 需求名称 | 需求描述 |
| --- | --- |
| 文件筛选 | 支持按股票代码、年份和文件类型筛选 10-K、10-Q、8-K。 |
| 文件正文 | 返回指定 SEC 文件的可阅读正文。 |
| 文件元信息 | 返回文件类型、报告期、公司名称、档案编号、注册地和雇主识别号。 |
| 关键章节 | 返回 Business、Risk Factors、MD&A 等主要章节内容。 |
| 财务摘要 | 返回净销售额、销售成本、毛利等核心数据。 |
| 历史财务数据 | 返回多年收入、成本、毛利、营业利润、净利润和每股收益。 |

## 社媒观点

| 需求名称 | 需求描述 |
| --- | --- |
| 股票相关帖子 | 按股票代码或公司名称获取相关社媒帖子。 |
| 排序方式 | 支持按热门和最新排序。 |
| 作者信息 | 返回作者名称、账号、头像标识。 |
| 帖子内容 | 返回正文、发布时间和相对时间。 |
| 互动数据 | 返回回复数、转发数、点赞数和浏览量。 |

## 页面聚合

| 需求名称 | 需求描述 |
| --- | --- |
| 个股看板数据 | 一次返回当前股票页面所需的概览、图表、财务和社媒数据。 |
| 空数据状态 | 当图表、财务或社媒数据不存在时，返回明确的空列表。 |
| 数据来源状态 | 返回各类数据是否获取成功，便于页面区分正常为空和获取失败。 |
