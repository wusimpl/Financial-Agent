# 项目计划

## 项目背景

这个项目要做一个面向美股的公司和股票分析智能体。

第一阶段先不做智能分析，只把信息源整理好，让系统可以稳定获取并展示公司财报、市场观点和股票走势。后面再基于这些信息源继续做分析能力。

## 当前目标

先把公司和股票分析所需的信息源打通，只展示信息，不做智能分析。

第一版只覆盖美股。

## 信息源计划

1. 财报数据

   使用 SEC 官方数据源，支持通过股票代码获取公司信息、10-K、10-Q、财报正文和结构化财务数据。

2. Twitter/X 观点

   使用现有 `twitter-cli`，后端只做统一包装，先支持搜索关键词和读取指定账号推文。

3. 股票价格走势

   使用现有 `easy-tdx` 获取美股 K 线，先作为辅助展示数据。

## 后端计划

后端先只做数据源接口：

- 每个数据源单独封装
- 统一做本地缓存
- 提供简单 API 给后续前端调用
- 暂时不做数据库、登录、任务队列和智能体逻辑

## 前端计划

前端当前只保留目录，暂时不开发。

后续再做一个简单页面，用来输入股票代码并展示三类信息源。

## 已完成

- 创建了后端项目结构
- 创建了前端占位目录
- 实现了本地文件缓存
- 接入了 SEC 财报数据源
- 支持按股票代码查询公司
- 支持查询 10-K / 10-Q 列表
- 支持按年份查询财报
- 支持按年份获取财报正文
- 接入了 Twitter/X 搜索和用户推文接口
- 接入了美股 K 线接口
- 添加了最小后端 API
- 添加了基础测试
- 用 AAPL 验证过财报、K 线和 Twitter 数据源

## 下一步

做一个聚合接口：

输入股票代码后，一次返回最近财报、指定年份财报、近期 K 线和相关 Twitter/X 内容。
